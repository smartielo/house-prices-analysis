### Project Overview
This project is a solution for the Kaggle competition: **"House Prices - Advanced Regression Techniques"**.
The goal is to predict the final price of each home in Ames, Iowa, using 79 explanatory variables describing (almost) every aspect of residential homes.

### Key Objectives
* **Data Cleaning:** Handle missing values and outliers.
* **Feature Engineering:** Transform variables to better suit the model.
* **Modeling:** Use Advanced Regression techniques (like Random Forest, XGBoost, or Lasso) to predict continuous values.
* **Metric:** Minimize the Root-Mean-Squared-Error (RMSE).

### Technologies Used
* **Python:** Main language.
* **Pandas & NumPy:** Data manipulation.
* **Matplotlib & Seaborn:** Data visualization.
* **Scikit-Learn:** Machine Learning models.
